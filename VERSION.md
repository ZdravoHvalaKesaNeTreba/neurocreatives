# Neurocreatives - История версий

## Версия 1.0 (17 марта 2026)

### 🎉 Первый релиз

Полнофункциональная платформа для анализа рекламных креативов из Telegram каналов с AI-анализом изображений.

### ✨ Основные функции

#### Парсинг и хранение
- Автоматический парсинг Telegram каналов
- Сохранение постов с изображениями в PostgreSQL
- Подсчет метрик вовлеченности (ER, просмотры, реакции)
- Управление списком каналов через веб-интерфейс

#### AI-анализ изображений
- Анализ креативов через OpenAI Vision API
- Определение типа креатива, сцены, объектов
- Распознавание эмоций и текста на изображении
- Оценка визуальной силы (1-10)

#### Веб-интерфейс

**Галерея:**
- Сеточное отображение креативов
- Фильтрация по каналам
- Детальная информация о каждом креативе
- Модальное окно с полной информацией

**Таблица:**
- Табличное представление всех креативов
- Фильтры по каналу, ER, просмотрам
- Сортировка по колонкам
- Скачивание изображений
- Копирование описаний для промптов
- Правая панель со статистикой

**Детальная панель:**
- Открывается по клику на креатив (в галерее или таблице)
- Полноэкранный backdrop с blur-эффектом
- Закрытие кликом вне панели или по кнопке X
- Вся информация о посте и AI-анализ

### 🐛 Известные ограничения

- Парсинг только публичных каналов
- Поддержка только изображений (без видео)
- Один AI-анализ на изображение (без повторного анализа)
- Фильтрация работает только на загруженных данных (без серверной пагинации)
