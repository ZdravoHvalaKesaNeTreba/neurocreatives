🚀 Основные функции

1. Галерея креативов

Визуальное представление всех собранных рекламных постов:

  • Карточки с превью изображения
  • Просмотры и вовлеченность (ER%) на каждой карточке
  • Клик на карточку открывает детальную информацию:
    • Полный текст поста
    • AI-описание изображения
    • Статистика (просмотры, пересылки, реакции, комментарии)
    • Ссылка на оригинальный пост в Telegram

2. Таблица

Табличное представление с возможностью сортировки:

  • Сортировка по каналу, дате, просмотрам, ER%
  • Колонка "Описание для промпта" — AI-анализ изображения
  • Кнопка "Открыть пост" — переход к оригиналу в Telegram
  • Кнопка "Детали" — открыть полную информацию

3. Фильтры

Настройте фильтрацию креативов:

  • По каналу — выбрать конкретный Telegram-канал
  • По ER% — от минимального до максимального значения
  • По просмотрам — задать диапазон
  • Подсказки: средний ER% и средние просмотры отображаются автоматически

4. Статистика

В боковой панели вы всегда видите:

  • Общее количество постов
  • Количество изображений
  • Сколько изображений проанализировано AI
  • Список всех каналов

⚙️ Настройки

Кнопка "Настройки" в правом верхнем углу открывает модальное окно с 4 вкладками:

Вкладка 1: Каналы

Управление списком Telegram-каналов для парсинга:

  1. Введите алиас канала (username без @)
  2. Каждый канал с новой строки
  3. Примеры: klientvsprav, trendach, sostav, durov
  4. Нажмите "Сохранить"

Парсер будет собирать посты из всех указанных каналов.

Вкладка 2: API ключ

Укажите ваш OpenAI API ключ для анализа изображений:

  1. Получите ключ на https://platform.openai.com/api-keys
  2. Вставьте ключ в формате: sk-...
  3. Нажмите "Сохранить"

⚠️ Без ключа анализ изображений работать не будет.

Вкладка 3: Промпт анализа

Настройте, что именно AI будет анализировать на изображениях.

Пример промпта:

Опиши это изображение для создания рекламного промпта.
Укажи:
- Сцену и обстановку
- Объекты на фото
- Цветовую палитру
- Эмоциональный посыл
- Наличие текста
- Тип креатива (продуктовая съемка, lifestyle, инфографика и т.д.)

Можно менять под свои нужды!

Вкладка 4: Планировщик

Настройте автоматический парсинг по расписанию.

Основные опции:

  • Включить планировщик — галочка для активации
  • Периодичность: ежедневно, еженедельно, раз в 2 недели, ежемесячно или выбрать дни недели
  • Глубина парсинга: только сегодняшние посты, за последние 3 дня или с определенной даты
  • Время запуска: выберите время в формате HH:MM (по МСК)
  • Срок действия: бессрочно или до определенной даты

🚀 Ручной запуск

В верхней панели есть 2 кнопки для ручного запуска:

📡 Запустить парсер

  • Собирает новые посты из всех настроенных каналов
  • Скачивает изображения
  • Сохраняет метаданные (просмотры, дата, текст)

🔍 Запустить анализ

  • Анализирует все непроанализированные изображения
  • Использует GPT-4o mini для описания
  • Сохраняет результаты в базу данных

📌 Примечание: Эти кнопки работают независимо от автоматического расписания.

💡 Полезные советы

🕐 Как часто парсить?

  • Для активных каналов: Ежедневно или каждые 3 дня
  • Для архивных задач: С определенной даты + разовый запуск

⏰ Оптимальное время запуска

  • Утро (08:00-10:00): Соберете посты за ночь
  • Вечер (20:00-22:00): Соберете посты за день

🎯 Работа с фильтрами

  1. Сначала посмотрите средние значения ER% и просмотров
  2. Установите фильтры выше среднего
  3. Используйте фильтр по каналу для анализа конкретного источника

🤖 AI-анализ

  • GPT-4o mini анализирует изображения пакетами по 20 штук
  • Между запросами задержка 1 секунда
  • Если видите ошибки 429 (rate limit) — это норма, система автоматически повторит позже

❓ Частые вопросы

1. Почему некоторые посты без изображений?

Старые посты могли быть собраны до добавления фильтра. Сейчас парсер не сохраняет посты без изображений. Если пост без картинки — показывается заглушка.

2. Сколько каналов можно добавить?

Без ограничений. Но помните:

  • Больше каналов = дольше парсинг
  • Рекомендуется: 10-30 каналов

3. Как часто обновляется статистика?

Статистика обновляется автоматически каждые 5 секунд.

4. Можно ли скачать креативы?

Да! Все изображения хранятся на сервере. Клик правой кнопкой мыши → "Сохранить изображение"

5. Что делать, если не работает парсер?

Проверьте:

  1. API ключи настроены? (Telegram API + OpenAI API)
  2. Каналы добавлены?
  3. Логи во вкладке "Логи" — там будет ошибка
