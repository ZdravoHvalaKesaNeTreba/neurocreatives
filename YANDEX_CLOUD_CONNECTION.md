# Как получить данные для подключения к PostgreSQL в Yandex Cloud

## Шаг 1: Откройте кластер PostgreSQL

1. Зайдите в консоль Yandex Cloud: https://console.cloud.yandex.ru
2. Выберите ваш каталог (folder)
3. В боковом меню выберите **Managed Service for PostgreSQL**
4. Найдите и откройте ваш кластер **postgresql790**

## Шаг 2: Получите данные подключения

### Вариант А: Через раздел "Обзор"

1. В открытом кластере перейдите на вкладку **"Обзор"**
2. Найдите раздел **"Подключение"** или **"Connection"**
3. Там будут данные:
   - **Хост** (Host): например `c-xxxxx.rw.mdb.yandexcloud.net`
   - **Порт** (Port): обычно `6432`
   - **Имя БД**: `db_neurocreatives`

### Вариант Б: Через раздел "Базы данных"

1. Перейдите на вкладку **"Базы данных"**
2. Найдите `db_neurocreatives`
3. Справа нажмите на три точки **⋮** → **"Получить строку подключения"**

### Вариант В: Через раздел "Пользователи"

1. Перейдите на вкладку **"Пользователи"**
2. Найдите пользователя (судя по скриншоту это `user1`)
3. Запомните имя пользователя
4. Если нужен пароль - создайте нового пользователя или сбросьте пароль

## Шаг 3: Соберите строку подключения

Формат строки подключения:

```
postgresql://USERNAME:PASSWORD@HOST:PORT/DATABASE?sslmode=require
```

Пример:

```
postgresql://user1:mypassword@c-xxxxx.rw.mdb.yandexcloud.net:6432/db_neurocreatives?sslmode=require
```

**Важно:** 
- Yandex Cloud требует SSL подключение, поэтому добавьте `?sslmode=require`
- Если порт не указан явно, скорее всего это `6432` (не стандартный 5432)

## Шаг 4: Что нужно скопировать

Вам нужны эти данные:

- 📍 **Host**: `c-xxxxx.rw.mdb.yandexcloud.net`
- 🔌 **Port**: `6432`
- 👤 **User**: `user1` (из скриншота видно)
- 🔑 **Password**: пароль пользователя user1
- 💾 **Database**: `db_neurocreatives`

## Где найти на скриншоте

На вашем скриншоте видно:
- Имя кластера: **postgresql790**
- Владелец БД: **user1**
- База данных: **db_neurocreatives**

Теперь нужно:
1. Кликнуть по кластеру **postgresql790** слева
2. Найти раздел подключения
3. Скопировать хост и порт
4. Узнать пароль от user1 (или создать нового пользователя)

## Готовая строка для config.py

После получения данных, строка будет примерно такой:

```python
DATABASE_URL = 'postgresql://user1:ВАШ_ПАРОЛЬ@c-xxxxx.rw.mdb.yandexcloud.net:6432/db_neurocreatives?sslmode=require'
```

Замените:
- `ВАШ_ПАРОЛЬ` - на реальный пароль
- `c-xxxxx` - на реальный хост из консоли
